# ☕️ Coffee House — Городская сеть кофеен

Современный лендинг для сети кофеен и кондитерских с онлайн-меню и адаптивной навигацией.

## 🚀 [Live Demo](https://IrinaSheshko.github.io/Coffee-House/)

---

### 🌟 Функционал и особенности:
* **Интерактивное меню:** Анимированное выпадающее меню для мобильных устройств (реализовано на GSAP).
* **Hover-эффекты:** Динамическое масштабирование изображений при наведении для улучшения UX.
* **Структурированный контент:** Четкое разделение на категории (торты, киши, десерты) и блоки преимуществ.
* **SEO-friendly:** Использование семантических тегов (`article`, `section`, `address`, `figure`).

---

### 🛠 Стек технологий:
* **HTML5 & CSS3:** Сложная верстка с использованием Flexbox и позиционирования.
* **JavaScript (GSAP):** Анимация мобильного меню и микро-взаимодействий.
* **Icons8:** Интеграция качественных иконок для визуализации услуг.
